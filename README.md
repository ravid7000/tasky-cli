# Tasky CLI

# Prerequisite
```sh
node >= 10.x
yarn >= 1.13.0
vscode >= 1.30
```

# Installation
Install dependencies using `npm` or `yarn` (recommended)

```sh
yarn
```
or
```sh
npm install
```

# Start `dev` server

```sh
yarn start
```
or
```sh
npm start
```

# Build Project

```sh
yarn build
```
or
```sh
npm run build
```
